JsMimicry
===========

Allows the use of Dart classes and objects in javascript.
Generates a special proxy classes.

Support:
 * Named constructors
 * Method
 * Optional positional parameters
 * Optional named parameters
 * Input parameter transform
 * Result mutation
 * Future (via result mutator)
 
Not support:
 * Factory
 * Field
 * Getter/Setter
 * Operator
